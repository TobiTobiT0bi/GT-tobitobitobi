Acerca de mí

👋 ¡Hola! Soy Tobias Coman, un aprendiz entusiasta del desarrollo de software con un gran deseo de crecer y aprender en este emocionante campo. Soy una persona creativa, con imaginación e iniciativa para desarrollar ideas nuevas o trabajar sobre ideas existentes. Tengo gran capacidad de trabajo, integración y flexibilidad. Busco desarrollarme profesionalmente dentro de un equipo de trabajo o de manera individual, dando lo mejor de mi, con muy buena predisposición y actitud hacia las responsabilidades que me sean asignadas. 🚀 Mis habilidades incluyen: » Creación de formularios y/o programas básicos con C# y Windows forms » Desarrollo de videojuegos con XNA y Monogame » Desarrollo web con Laravel haciendo uso de estilos Tailwind CSS o Bootstrap 5 » Diseño responsive para garantizar una experiencia óptima en todos los dispositivos. » Conocimientos en HTML, CSS y JavaScript para crear interfaces de usuario dinámicas. » Integración de tecnologías back-end si es necesario para la funcionalidad del proyecto.
